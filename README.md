## 🐘 Day 1 亮点详解：象山 (Elephant Mountain) — 台北101经典打卡

朋友力荐的象山，是拍台北101经典明信片角度的最佳地点！登顶仅需15-20分钟，非常轻松。

- **🚇 交通：** 捷运淡水信义线(红线)至**"象山站"(Xiangshan)**，2号出口出站
- **🚶 步行：** 沿公园旁步道步行约650m(约20分钟)到达登山口
- **🥾 登山：** 约15-20分钟到顶，全程阶梯。**注意在岔路口左转！** 约8-10分钟后有平台(含洗手间)，此处分岔多条路
- **📸 最佳时间：** 傍晚前往，可同时拍到白天、夕阳与夜景的台北101。岩石上的拍照位是经典机位
- **📍 周边：** 象山就在台北101/信义商圈旁边，从山脚步行至101仅15分钟。可结合101观景台一起玩
- **🍜 附近夜市：** 通化夜市(临江街) — 从山脚步行25-30分钟，或搭捷运红线2站到信义安和站。也可回市区去宁夏夜市

> **💡 提示：** 到达当天是红眼航班清晨6:15抵达，建议上午到酒店寄存行李后补觉休息，下午3-4点再出发去象山，时间刚好。

## 🔑 Day 2 亮点详解：阳明山文化大学后山夜景

文化大学位于台北市北部阳明山上，与故宫博物院(士林区)同方向，加进行程完全顺路。

- **🌅 上午：** 故宫博物院 — 探索中华文化瑰宝(约3小时)
- **☕ 下午：** 大稻埕/迪化街 — 老台北复古风情，老宅改造咖啡厅喝手冲
- **🌃 傍晚至晚上：** 阳明山文化大学后山"情人坡" — 无遮挡俯瞰整个台北盆地，台北101尽收眼底

**🍽️ 景观餐厅推荐(情人坡下方，依山而建)：**
| 餐厅 | 特色 |
|---|---|
| **屋顶上 (The Top)** | 台北最著名夜景餐厅之一，巴厘岛热带度假风情，露天沙发区，不能预订需现场排队 |
| **草山夜未眠 (Sleepless in Yangmingshan)** | 绝佳视野，座位浪漫，台式热炒+西式简餐，氛围极放松 |

**🚗 交通：**
- **上山：** 大稻埕逛完直接叫Uber(定位"文化大学后山情人坡"或具体餐厅)，车程约30-40分钟。或捷运到"剑潭站"转"红5"公交直达文化大学，跟着学生人群走。
- **下山：** 餐厅前台帮叫出租车，或自己叫Uber。下山后可顺道去士林夜市吃宵夜。

## 🛂 签证/入台证信息 (Visa / Entry Permit)

### 适用类别
从第三地(阿姆斯特丹)出发的大陆居民，属于**"第三类来台观光"**（大陆地区人民来台从事观光活动，旅居海外者）。

### 停留天数：15天（翌日起算）
> 官方规定：**自入境之翌日起不得逾15日**（台湾移民署）
> 即：入境当天不算，从**第二天**开始数15天。

| 项目 | 说明 |
|---|---|
| 入境日 | 9/19 (不计入) |
| 第1天 | 9/20 |
| 第15天(最后一天) | **10/4** ✅ |
| 离境航班 | 10/4 11:40 高雄→南京 |

**✅ 行程完全符合15天限制！** 10/4是第15天(最后合法停留日)，当天11:40航班离境合法。

### ⚠️ 注意事项
- **零缓冲：** 10/4已是最后一天，若航班取消/延误到10/5将构成**逾期停留**，后果严重。强烈建议购买含航班延误/取消保障的旅行保险。
- **从第三地出发无需大通证：** 从阿姆斯特丹(非大陆境内)出发，不需要"大陆居民往来台湾通行证(大通证)"，仅需**入台证**。
- **申请方式：** 可线上申请入台证（第三类），建议提前办理。
- **官方来源：** 台湾移民署 https://www.immigration.gov.tw
